# Hello World

Een eenvoudige "Hello World" webpagina ontwikkeld in HTML.

## Beschrijving

Dit project bevat een basis HTML-webpagina die "Hello World!" toont. Het is een minimalistisch voorbeeld dat kan dienen als startpunt voor het ontwikkelen van webapplicaties.

## Bestandsstructuur

```
├── index.html          # Hoofd HTML-bestand
├── README.md           # Dit README-bestand
└── .gitignore          # Git ignore-configuratie
```

## Gebruik

1. Kloon de repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigeer naar de projectmap:
   ```bash
   cd <project-map>
   ```

3. Open `index.html` in uw webbrowser of gebruik een lokale webserver.

## Vereisten

- Een moderne webbrowser (Chrome, Firefox, Safari, Edge)
- Optioneel: Een lokale webserver voor ontwikkeling

## Technologieën

- HTML5
- CSS (geen externe stylesheets gebruikt)

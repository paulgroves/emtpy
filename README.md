# Hola Byd

Mae hwn yn ymarfer HTML syml sy'n arddangos tudalen "Hello World" yn y porwr.

## Disgrifiad

Mae'r prosiect hwn yn cynnwys ffeil HTML sylfaenol sy'n creu tudalen we syml gyda'r testun "Hello World!". Mae'n enghraifft dda o ddechrau gyda datblygu gwe HTML.

## Ffeiliau

- `index.html` - Prif ffeil HTML sy'n cynnwys y strwythur tudalen
- `.gitignore` - Ffeil ffurfweddu i eithrio ffeiliau system a golygyddion rhag cael eu commitio

## Sut i Ddefnyddio

1. Clonio'r repositori:
   ```bash
   git clone <url-reposi>
   cd <enw-reposi>
   ```

2. Agor `index.html` mewn porwr gwe:
   - Dbl-glicio ar y ffeil
   - Neu defnyddio gweinydd lleol fel `live-server`
   - Neu lwytho'r ffeil yn uniongyrchol i'ch porwr

## Technolegau

- HTML5
- Dim CSS na JavaScript wedi eu cynnwys (yn unig HTML syml)

## Gosodiad

Nid oes angen gosodiad ar gyfer y prosiect hwn. Mae'n ffeil HTML sefydlog y gellir ei hagor yn uniongyrchol mewn porwr gwe.

# Croeso

Mae'r prosiect hwn yn cynnwys tudalen we statig syml sy'n dangos y neges "Hello World!". Fe'i bwriedir fel man cychwyn i ychwanegu mwy o gynnwys neu arddulliau yn nes ymlaen.

## Sut i ddefnyddio

1. Agorwch y ffeil `index.html` yn eich porwr gwe dewisol.
2. Dylech weld y pennawd "Hello World!" yn cael ei rendro ar y dudalen.

## Cysylltu

Os hoffech gyfrannu neu os oes gennych gwestiynau, crëwch docyn newydd neu gyflwynwch eich syniadau drwy'r sianel arferol y tîm.
